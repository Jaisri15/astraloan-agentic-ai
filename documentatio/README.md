#  Agentic AI Loan Underwriting System

This project is a **hackathon-ready Agentic AI system** that evaluates loan applications using:

- Credit Score  
- Income  
- Debt-to-Income Ratio  
- Employment Type  
- Automated Risk Scoring  
- XAI (Explainable AI) reasoning  


#  Purpose
Traditional loan evaluation is slow, biased, and often non-transparent.  
This system speeds up decision-making while giving **clear explanations** of every approval/rejection.


#  Key Features
- Automated risk scoring  
- Transparent reasoning for every decision  
- Bias reduction  
- Fast processing  
- Clean UI + Working Backend  
- Perfect for demo and judging
