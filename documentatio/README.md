# Healthcare Monitoring App

This is a demo application designed for hackathons and project presentations.  
It contains:

## ✔ Features
- Login page (Parameter ID, Wi-Fi/Bluetooth, Location)
- Dashboard showing:
  - Sugar level
  - BP level
  - SO2
  - ECG
  - Oxygen level
- AI Suggestions for abnormal parameters
- Video call mock feature
- Basic backend using Node.js + Express
